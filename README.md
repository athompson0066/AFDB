# AFDB & Google Device Financing Slideshow

An interactive, professional 41-slide presentation platform built for the **Google and AFDB Affordable Device Financing Model Project**. This application serves as a strategic advisory framework for bridging the digital usage gap in Sub-Saharan Africa, with a specific focus on the Nigeria pilot program.

## üöÄ Key Features

- **Interactive 2-Column Layout**: High-impact, contextually relevant tech visuals paired with data-driven strategic content.
- **3D Slide Transitions**: Smooth, performant transitions using CSS 3D transforms (translateX and rotateY).
- **AI-Powered Advisor**: Integrated ChatBot featuring a virtual representation of **Linda Quaynor** (Project Lead) to answer project-specific inquiries using the Google Gemini API.
- **Dynamic Data Visualizations**: Custom-built responsive bar charts, metric tables, and progress indicators for economic and DDI (Daily Disposable Income) analysis.
- **Authoring Suite**: Password-protected "Admin Mode" to add, duplicate, reorder, or edit slides in real-time without touching the code.
- **Persistence**: Local storage integration (v4) to save presentation edits across browser sessions.
- **Ken Burns Effects**: Cinematic background animations for a premium presentation feel.
